## рҹҺҜ Project Objective

**AI-Mock-Mentor** is a full-stack, AI-powered interview simulation platform built for:

- рҹҺ“ **Students** preparing for campus placements
- рҹ’ј **Job Seekers** targeting product-based and service-based companies
- рҹ§  **Developers** grinding for FAANG/MAANG-level technical interviews
- рҹ“Ҳ **Professionals** leveling up their communication and leadership interview skills

It doesn't just ask questions вҖ” it *thinks*, *adapts*, and *coaches* you like a real human interviewer would.

## вңЁ Features

<details>
<summary><b>рҹ§  1. Smart Adaptive AI Interviewer</b></summary>

<br/>

- **Dynamic Difficulty Scaling** вҖ” The AI analyzes each response and adjusts the complexity of the next question accordingly.
- **Domain-Specific Tracks** вҖ” Choose from:
  - рҹ’» Software Development
  - рҹ“Ҡ Data Science & ML
  - рҹҸў HR & Behavioral
  - рҹ“ҡ MBA & Group Discussion
- **HR + Technical Mix** вҖ” Simulates real-world interview rounds with a blend of soft-skill and hard-skill questions.

</details>

<details>
<summary><b>рҹ“„ 2. Resume-Based Personalized Questions (Advanced)</b></summary>

<br/>

- **Upload your resume** (PDF/DOCX) and the AI parses it to extract your skills, projects, and experience.
- **Hyper-Personalized Questions** вҖ” If you listed Python and a web-scraping project, expect:
  - *"Walk me through the architecture of your web scraper."*
  - *"How did you handle rate-limiting and anti-bot mechanisms?"*
- Every interview is **uniquely yours** вҖ” no two sessions are identical.

</details>

<details>
<summary><b>рҹ“Ҡ 3. AI Feedback & Performance Analytics</b></summary>

<br/>

After every session, receive a **comprehensive performance report** with:

| Metric | What It Measures |
|--------|-----------------|
| рҹ—ЈпёҸ Communication Score | Clarity, pace, vocabulary, and filler word frequency |
| рҹ§Ә Technical Accuracy | Correctness and depth of technical answers |
| рҹ’Ә Confidence Score | Vocal tone, hesitation, and eye contact (video mode) |
| вӯҗ Overall Rating | Weighted composite score out of 10 |

**Actionable Feedback Examples:**
- *"You used 'um' 14 times. Practice pausing instead of filling silence."*
- *"Your answer lacked depth вҖ” explain time complexity next time."*
- *"Maintain eye contact with the camera to project confidence."*

</details>

<details>
<summary><b>рҹ’» 4. Coding Interview Mode рҹ”Ҙ (Killer Feature)</b></summary>

<br/>

A full **in-browser technical assessment environment**:

- **Integrated Code Editor** вҖ” Syntax highlighting, auto-complete, multi-language support
- **Question Bank** covering DSA, SQL Queries, and Python Scripting
- **Auto Code Evaluation** вҖ” AI reviews your solution for correctness, efficiency, and code style
- **Live Timer** вҖ” Simulates real interview pressure with configurable time limits
- **Hints System** вҖ” Reveal progressive hints without spoiling the full solution

</details>

## рҹӣ пёҸ Tech Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| **Frontend** | React.js + Tailwind CSS | Responsive, interactive UI |
| **Backend** | Node.js + Express.js | RESTful API, routing, auth |
| **AI Engine** | FastAPI / Flask (Python) | LLM integration, NLP processing |
| **LLM** | OpenAI GPT-4 API | Question generation, feedback analysis |
| **Voice Analysis** | Web Speech API + NLP | Filler word detection, pacing metrics |
| **Video Analysis** | OpenCV + MediaPipe | Facial expression & confidence scoring |
| **Database** | MongoDB / Firebase | User data, session history, analytics |
| **Auth** | JWT + OAuth 2.0 | Secure user authentication |

## рҹҡҖ Getting Started

### вң… Prerequisites

```bash
node >= 18.0.0
npm  >= 9.0.0
python >= 3.10
mongodb >= 6.0
```

### рҹ“Ұ Installation

**1. Clone the repository**
```bash
git clone https://github.com/AnuragKumar0429/ai-mock-mentor.git
cd ai-mock-mentor
```

**2. Setup the Frontend**
```bash
cd client
npm install
cp .env.example .env
npm run dev
```

**3. Setup the Backend (Node.js)**
```bash
cd server
npm install
cp .env.example .env
npm run dev
```

**4. Setup the AI Service (Python)**
```bash
cd ai-service
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
uvicorn main:app --reload
```

**5. Environment Variables**

```env
# Frontend (.env)
VITE_API_BASE_URL=http://localhost:5000
VITE_AI_SERVICE_URL=http://localhost:8000

# Backend (.env)
PORT=5000
MONGODB_URI=mongodb://localhost:27017/aimockmentor
JWT_SECRET=your_jwt_secret_here
OPENAI_API_KEY=your_openai_api_key_here

# AI Service (.env)
OPENAI_API_KEY=your_openai_api_key_here
ALLOWED_ORIGINS=http://localhost:5173
```

Open [http://localhost:5173](http://localhost:5173) and start your mock interview! рҹҺү
